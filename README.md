# Football table react/redux web app 
### User can add football match result and watch the final table below
### Teams can meet each other only two times per season

## Scoring by average football rules: 
 #### - win: 3pts;
 #### - loose: 0pts;
 #### - draw: 1pts;

##### if the teams have the same number of points by the end of the season, then the difference between goals scored and goals conceded is considered
